
# Booking website

This is a website project for hotel room reservations.


## Run Locally

Clone the project

```bash
  git clone https://github.com/chutrinh/booking-react.git
```

Go to the project directory

```bash
  cd booking-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
